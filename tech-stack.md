# Tech Stack

## Frontend
- **React 19** with **TypeScript**
- **React Router** — client-side routing
- **Tailwind CSS** — styling
- **shadcn/ui** — component library

## Backend
- **Node.js** with **Express** and **TypeScript**

## Database
- **PostgreSQL** — primary data store
- **Prisma** — ORM

## Authentication
- Database sessions (session stored in PostgreSQL via Prisma)

## AI
- **Claude API (Anthropic)** — ticket classification, summaries, and suggested replies

## Email
- **SendGrid** — sending replies and receiving inbound emails via Inbound Parse webhook

## Deployment
- **Vercel** — frontend
- **Neon** — managed PostgreSQL
